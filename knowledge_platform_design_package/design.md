# 知识库贡献平台 UI 设计说明

版本：v1.0  
生成时间：2026-06-04 10:34:25  
适用项目：企业知识库文件贡献与 RAGFlow 同步平台

---

## 1. 设计定位

本设计用于公司内部知识库贡献平台，核心目标是让员工可以通过 Web 页面上传文档，管理员可以完成审核、分类、AI 分析配置、RAGFlow 同步和统计分析。

整体定位：

```text
企业级后台管理系统
+
知识库文档贡献入口
+
RAGFlow 同步管理后台
+
AI 文档治理配置中心
```

## 6. 交互与状态规范

### 6.1 文件状态

建议状态：

```text
uploaded                 已上传
pending_review            待审核
approved                  已审核
rejected                  已拒绝
queued                    等待同步
syncing                   同步中
uploaded_to_ragflow       已上传到 RAGFlow
parsing                   RAGFlow 解析中
parsed                    解析完成
failed                    失败
disabled                  已禁用
deleted                   已删除
```

### 6.2 审核状态颜色

```text
待审核：橙色
审核中：蓝色
已通过：绿色
未通过：红色
已拒绝：红色
```

### 6.3 同步状态颜色

```text
未同步：灰色
待同步：灰色/蓝色
同步中：蓝色
已同步：绿色
同步失败：红色
```

### 6.4 风险等级颜色

```text
低风险：绿色
中风险：橙色
高风险：红色
严重风险：深红色
```

## 8. 设计落地注意事项

### 8.1 生成图仅作高保真方向参考

本包中的图片是页面视觉方向和布局参考，真实前端开发时需要根据实际字段、接口、权限和业务状态进一步细化。

### 8.2 中文文案可以调整

设计图中的中文文案可作为默认文案，开发时可以根据公司内部术语替换。

### 8.3 页面需要响应式适配

虽然当前设计主要是桌面端，但建议至少适配：

```text
1440px+
1366px
1280px
```

内部后台优先保证桌面端体验。

### 8.4 管理员页面信息密度较高

文件管理、统计分析、用户管理页面信息较多，建议开发时提供：

- 表格横向滚动
- 固定表头
- 固定操作列
- 列配置
- 高级筛选收起/展开

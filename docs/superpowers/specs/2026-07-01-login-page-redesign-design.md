# 登录页面重新设计

## 目标

将登录页面从信息密集的功能展示页改为视觉温暖、品牌突出的认证入口。保持项目蓝色主题一致性，用纯 CSS 装饰替代当前的功能卡片堆砌。

## 设计决策

| 维度 | 现有 | 新设计 |
|---|---|---|
| 左侧内容 | 标题 + 状态条 + 4 特性卡 + 仪表盘预览 | 深蓝渐变背景 + 毛玻璃光球 + 品牌标识 |
| 右侧表单 | 白色区域内左对齐的表单 | 浅灰背景上悬浮居中的白色卡片 |
| 视觉气质 | 信息密集、无焦点 | 温暖亲和、品牌突出 |
| 装饰元素 | 两个透明圆 | 三个毛玻璃光球 + 缓慢浮动动画 |
| 输入框 icon | MailOutlined / LockOutlined 前缀 | 无前缀，更简洁 |

## 左侧背景

### 渐变底色

```css
background: linear-gradient(135deg, #1a3a6e 0%, #2563EB 50%, #7C93E0 100%);
```

深蓝到柔蓝灰的 135° 渐变，用深色衬托右侧白色卡片。

### 毛玻璃光球

3 个大尺寸圆形，绝对定位，使用 `radial-gradient` + `filter: blur(60-80px)`：

| 球 | 位置 | 尺寸 | 颜色 |
|---|---|---|---|
| 1 | 右上角偏移 | 350px | `rgba(99, 102, 241, 0.3)` 蓝紫 |
| 2 | 中间偏右 | 280px | `rgba(56, 189, 248, 0.25)` 天蓝 |
| 3 | 左下角 | 320px | `rgba(59, 130, 246, 0.35)` 蓝色 |

### 浮动动画

```css
@keyframes auth-float-1 {
  0%, 100% { transform: translate(0, 0); }
  50% { transform: translate(30px, -20px); }
}
```

每个球不同的周期（15s / 18s / 20s）和方向，`ease-in-out`。`prefers-reduced-motion: reduce` 时关闭。

### 品牌标识

左下角，`position: absolute; bottom: 48px; left: 48px`：

- 40px 白色圆角方块内的 DatabaseOutlined 图标（白底蓝 icon 反色为白色）
- 平台名「知识库贡献平台」白色 16px 粗体
- 下方标语「让企业知识持续沉淀与同步」白色半透明 14px

## 右侧表单卡片

### 卡片容器

- 背景：`#FFFFFF`
- 圆角：`20px`
- 阴影：`0 20px 60px rgba(0, 0, 0, 0.12)`
- 最大宽度：`440px`
- 内边距：`40px`
- 垂直水平居中于右侧区域
- 右侧区域背景：`#F8FAFC`

### 表单元素（从上到下）

1. **品牌图标**：40px 蓝色圆角方块（`#1677FF` 背景 + 白色 DatabaseOutlined），居中
2. **标题**：「欢迎回来」，24px，`#0f172a`，`font-weight: 700`，居中
3. **副标题**：「使用公司邮箱登录知识库工作台」，14px，`#667085`，居中，`margin-bottom: 32px`
4. **邮箱输入框**：label「公司邮箱」，高度 48px，圆角 10px，无前缀 icon，placeholder「name@company.com」
5. **密码输入框**：label「密码」，高度 48px，圆角 10px，无前缀 icon
6. **辅助行**：「记住我」复选框（左） +「忘记密码」链接（右），`margin-bottom: 24px`
7. **登录按钮**：48px 高，圆角 10px，`#1677FF`，全宽，`font-weight: 600`
8. **注册链接**：「还没有账号？ 立即注册」，居中，`margin-top: 24px`

### 输入框聚焦态

```css
border-color: #1677FF;
box-shadow: 0 0 0 3px rgba(22, 119, 255, 0.1);
```

## 响应式

### < 1024px（平板和手机）

- 隐藏左侧背景区域
- 右侧占满全屏，背景简化为浅蓝渐变 `linear-gradient(135deg, #EEF5FF, #F8FAFC)`
- 表单卡片居中，保留阴影和圆角
- 左上角显示小品牌标识（logo + 名称）

### < 640px

- 卡片内边距缩小为 `24px`
- 卡片圆角改为 `16px`

## 动效

| 元素 | 效果 | 时长 | 触发 |
|---|---|---|---|
| 光球浮动 | translateX/Y 循环 | 15-20s | 持续 |
| 卡片进入 | opacity 0→1 + translateY 20→0 | 300ms | 页面加载 |
| 输入框聚焦 | 蓝色边框 + 光晕 | 200ms | focus |
| 按钮 hover | 亮度微增 | 150ms | hover |

所有动效在 `prefers-reduced-motion: reduce` 时禁用。

## 改动范围

| 文件 | 改动 |
|---|---|
| `frontend/src/pages/AuthLayout.tsx` | 大幅精简：删除 authFeatures、authStatusItems、auth-preview 等所有展示组件。左侧只保留品牌标识 + 标语。右侧改为悬浮卡片。 |
| `frontend/src/styles.css` | 重写 `.auth-*` CSS（约 350 行）：新增渐变背景、光球、卡片阴影样式，删除所有 feature/status/preview 样式。 |
| `frontend/src/pages/Login/index.tsx` | 小改动：去掉 MailOutlined/LockOutlined 前缀，标题改为「欢迎回来」。 |

### 不影响的部分

- 主题 token（`tokens.ts`、`antd-theme.ts`）
- 注册页 / 忘记密码 / 重置密码（共用 AuthLayout，自动继承新样式）
- 其他业务页面
- API 调用逻辑
- StatusTag 组件（删除 AuthLayout 中的引用，组件本身保留）

## 注册页面同步

注册页使用同一个 AuthLayout，自动获得新的背景和卡片样式。注册表单的 `auth-form-grid`（两列布局）保持不变。
